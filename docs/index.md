# The Hacked Semester

Welcome to **The Hacked Semester** – a practical course on developer tools and workflows, inspired
by [MIT The Missing Semester](https://missing.csail.mit.edu/) and [CSDIY Wiki](https://csdiy.wiki).
This course also draws insights from [NJU OS Course by Yanyan Jiang](https://jyywiki.cn) and
[Linux 101 by USTC LUG](https://101.lug.ustc.edu.cn).

## What's this about?

Computer science classes teach advanced topics like algorithms, operating systems, and machine
learning, but often skip the essential tools that make daily work efficient and enjoyable. This
course fills that gap by covering the practical skills every developer needs:

- **Where to live?** – Setting up your development environment
- **Where to work?** – Essential tools for coding and collaboration
- **Where to deliver?** – Building and debugging your projects
- **Where to run?** – Containerization and virtualization
- **Where to talk?** – Documentation and communication tools

## Quick Start

1. **Browse the topics** using the navigation menu
2. **Follow along** with examples in the `examples/` directory
3. **Practice** with exercises in the `exercises/` directory
4. **Set up your environment** using scripts in `scripts/`

## Course Philosophy

This course is "hacked" together out of intolerance for low-quality resources. It represents how I
personally use and hack the tools in my workflow – with **NO WARRANTY**, as most licenses would
disclaim.

### Why "The Hacked Semester"?

Because it's indeed hacked. I don't believe in boasting about one's own course. The goal is simply
to produce an 80-out-of-100 course that's practical, honest, and useful.

### Missing something?

If a tool or topic you expect isn't here:

1. Check if it was omitted by design (e.g., some tools are excluded intentionally)
2. If you believe it should be included, open an issue with **evidence-based arguments** and
   **without personal criticism**
