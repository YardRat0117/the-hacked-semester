# Exercises

This directory contains practical exercises for each topic in The Hacked Semester.

## Structure

Exercises are organized by category:

- `live/` - Terminal, shell, WSL, Linux, SSH
- `work/` - Vim, LSP, Git, GitHub/GitLab
- `deliver/` - Make, CMake, GDB
- `run/` - Docker, Virtual Machines
- `talk/` - Markdown, LaTeX

## How to Use

1. Read the corresponding topic documentation
2. Attempt the exercises without looking at solutions
3. Check your work using provided test scripts (if available)
4. Review solutions (if provided) to compare approaches

## Note on Solutions

Solutions may be provided in a separate branch or repository to encourage independent problem
solving.
