# Terminal & Shell Exercise 1: Basic Navigation

## Objective

Practice basic terminal navigation and file operations.

## Tasks

1. **Navigation**
   - Open a terminal
   - Navigate to your home directory
   - Create a directory called `hacked-semester-practice`
   - Navigate into that directory

2. **File Operations**
   - Create a file called `notes.txt`
   - Add the text "Learning terminal commands" to the file
   - Create a copy called `notes-backup.txt`
   - Rename `notes.txt` to `journal.txt`

3. **Text Processing**
   - Create a file `numbers.txt` with numbers 1 through 10 (one per line)
   - Count how many lines are in the file
   - Display only even numbers from the file

4. **Cleanup**
   - Remove all files created in this exercise
   - Remove the `hacked-semester-practice` directory

## Verification

Run the provided test script to check your work:

```bash
./test-exercise1.sh
```

## Tips

- Use `man` command to read manual pages (e.g., `man ls`)
- Press Tab for auto-completion
- Use `history` to see previous commands
