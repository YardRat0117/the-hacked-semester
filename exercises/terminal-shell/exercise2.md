# Terminal & Shell Exercise 2: Shell Scripting

## Objective

Learn basic shell scripting concepts.

## Tasks

1. **Simple Script**
   - Create a script called `greet.sh`
   - Make it print "Hello, [your name]!"
   - Make the script executable
   - Run the script

2. **Interactive Script**
   - Create a script called `calculator.sh`
   - Ask user for two numbers
   - Perform addition, subtraction, multiplication, and division
   - Display all results

3. **Function Practice**
   - Create a script with a function `create_backup`
   - The function should take a filename as argument
   - It should create a backup with timestamp: `filename.backup.YYYYMMDD`
   - Test with different files

4. **Error Handling**
   - Modify `calculator.sh` to handle division by zero
   - Add error message when divisor is zero

## Verification

Test your scripts with different inputs to ensure they work correctly.

## Advanced Challenge

Create a script that organizes files in a directory by extension (e.g., move all `.txt` files to
`txt/` folder).
