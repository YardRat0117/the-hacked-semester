# Examples

This directory contains code examples for each topic in The Hacked Semester.

## Structure

Examples are organized by category:

- `terminal-shell/` - Shell scripts, command examples
- `git/` - Git workflows, commands, configurations
- Additional directories will be added for other topics

## How to Use

1. Browse examples related to the topic you're learning
2. Run examples to see them in action
3. Modify examples to experiment with variations
4. Use examples as templates for your own work

## Running Examples

Most examples are simple scripts that can be run directly:

```bash
# Make scripts executable
chmod +x example.sh

# Run the script
./example.sh
```
